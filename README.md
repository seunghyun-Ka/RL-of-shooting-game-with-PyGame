# Project name : RL-of-shooting-game-with-PyGame
## 프로젝트 소개
1. 파이게임으로 슈팅게임을 제작
2. 슈팅 게임을 RL(강화학습)으로 학습
3. 강화학습이 적용된 게임을 플레이

# Project information
## Requirement

2. anaconda
3. tensorflow
4. numpy
